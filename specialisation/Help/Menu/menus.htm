Archetype Editor Menus

The editor has a main menu and a lot of context menus that are available by right clicking the mouse.

Main menu

The main menu consists of:
  • The File menu
    • Open - to open an existing archetype
    • New - to create a new archetype
    • Save as - to save an archetype
    • Specialise - to specialise an archetype
    • Exit - to close the application
  • The View menu - which allows viewing of a Protege knowledge base of proto-archetypes
  • The Publish menu - which is not enabled
  • The Language menu
    • Available languages - moves to the Languages and Terminologies screen
    • Add a language
    • Change language - to change the display language of the archetype.
  • The Terminology menu
    • Available terminologies - moves to the Languages and Terminologies screen
    • Add a terminology
  • The Help menu
    • Help Topics - which display this help file
    • Report an issue - opens the bug and issue reporting page (Internet)
    • Licence - which displays licence information
    • About Ocean Informatics - information about the company that have developed this tool (Internet)
    • About the Ocean Editor - displays the start up splash screen including the version number
