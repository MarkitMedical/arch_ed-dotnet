Change the data type of an element

The editor allows you to change the data type of an element. Press the button on the left of the structure and choose the new data type from the menu. The editor offers a range of data types:

 Text which can be coded or free text, internal terms or from an external terminology
Quantity a real number and units
Count an integer with no units
Date/Time a date and/or time
Ordinal a number and text as pairs
Boolean true of false
Any which means any data type can be substituted at runtime

You are NOT able to change the data type when an archetype has been specialised - as the data type must be consistent when specialised - and so the button will not be visible.
