Add a terminology binding to a term

Adding a term binding is done at any stage of development of the archetype. The process links, with conditions if necessary, a data point in the archetype with a code in a terminology. To add a binding, the terminology to which you want to bind the term must already be added.
  • Move to the Terminology Tab and  choose Term Bindings

  • Now choose the terminology to which you want to make the binding from the drop down list

If the terminology is not on the list you must add a terminology.
  • Bindings already represented in an archetype are denoted in the 'tree view' of the archetype by the symbol to the left of the bound data point.
  • To add a binding select a data point or term by clicking on it within the tree view - then move to the bottom of the screen.

  • Press the button to add a code from the terminology you have chosen.

  • The binding can be to the node (in which case it will be repeated at all points in the data that this node appears) or to the full path within the archetype (when it will only relate to data at this point). Use the radio button to choose between the two ( a P will appear to the right of the Binding in the code list if you choose path, and an N if you choose the node).
  • Add the new code and press OK.

  • The new code will appear in the Bindings Code list.
  •  If a node is repeated in the archetype and you have chosen to bind to that node (rather than a the full path) you will see that the binding symbol is displayed by all instances of the node. If you bind to the full path, then it will only show by that particular instance.
