View the archetype in a particular language


You can use the Language menu or the Languages and terminology screen to change the display language of the archetype. If the language you require is not available you need to translate the archetype.
The display language of the archetype is a representation of the archetype in a particular language - importantly there is no language primacy ie terms are represented by surrogates which are translated into multiple languages.
You can safely add a language to the archetype and have it reviewed by another.
