Translate an archetype

Translating an archetype involves adding another language to the archetype and then translating all terms associated with the codes.

To translate the terms efficiently, choose the 'Terminology' Tab, and the 'Terms' Tab beneath it.

Now change to the new language using Menu > Language > Change and choose the new language from the menu.

This will change the terms to present the terms as translated from the primary language of the archetype. Each untranslated term then has an '*' pre-pended and the primary language code in brackets (in this case (en)) appended.

It is necessary to do the same for the description of the constraint codes used in the archetype. Choose the 'Constraints' Tab below the 'Terminology' Tab and follow the same process.

Now click on each term and description and type in the appropriate translation in the new language. Then save the archetype.
