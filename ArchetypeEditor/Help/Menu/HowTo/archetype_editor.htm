Archetype Editor

Supported by the Australian General Practice Computing Group and Commonwealth Department of Health and Aging.

An openEHR tool to create archetypes,

The Archetype editor is an openEHR (open source) tool that enables an expert clinician to build archetypes. These artefacts may be shared and used by systems to check the validity and completeness of data. You can use the help as a reference or may press F1 while using the application.

How to:

  • Design an archetype
  • Create a new archetype
  • Edit an existing archetype
  • Download an archetype
  • Specialise an archetype
  • Translate an archetype
  • Bind a term to an external terminology
