Adding criteria to a binding

A binding may require a crieria to be added - such as a LOINC code for a temperature measured in the ear. The binding of the data is to the temperature measurement, but only when the site of measurement was in the ear.
  • Open the Temperature archetype
  • Go to the Terminology > Term bindings Tab
  • Now click on the Temperature that appears under the 'Any Event' path. There are lots of paths in this archetype to represent all the LOINC codes and enable importing of LOINC data into the EHR.


The code appears in the Bindings Code list. Now add a new code as described in adding a term binding.


  • When the new code has been added press the button to the right of the code list - in the criteria list. The criteria screen will appear.

  • The criteria displayed are criteria that can be set for the node clicked in the 'tree view' above - in this case the temperature. Click on Protocol/Site of measurement to set where the temperature was measured.

  • Choose Aural from the drop down list and press OK.
You have now added a criteria to a term binding.
