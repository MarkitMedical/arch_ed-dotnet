Add or remove a cluster

A cluster is a node in a tree and is only available in a tree structure.

To add a cluster, follow the instructions for adding an element but choose a cluster from the popup menu after pressing the button or dragging the icon onto the list, tree or table.

To remove a cluster you can:

  • click on the cluster to be removed and press the button;
  • right click on the cluster and choose 'Remove..' from the pop-up menu; or
  • click on the cluster and press the delete button.

You will be asked to confirm that this is what you want to do - press OK if it is, Cancel if not.
