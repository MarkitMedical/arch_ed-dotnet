Specialise an element

Once an archetype is specialised an element within it can be specialised. This option is only available on the 'right-click' or context menu when the list item, the node of a tree or row of a table is selected.

You need to confirm this specialisation. The specialised element replaces the parent element in the tree or list.

.

Then rename the specialised element which has a "!" prefix to ensure that the need to rename is noticed. Specialised elements have a popup window which shows the specialisation hierarchy of the terms.
