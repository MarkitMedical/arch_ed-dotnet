Edit the header of an archetype

Concept and description

The header contains the text describing the concept represented by the archetype and a more detailed description.

Subject of care

You may also restrict the subject of care to which this archetype may apply.

Subjects added may be removed using the [-] button.

Specialisation hierarchy

If the archetype has been specialised, the header will display the parent(s) of the current archetype as a hierarchy.
