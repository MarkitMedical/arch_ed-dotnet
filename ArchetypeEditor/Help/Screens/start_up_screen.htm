Start up screen

The start up screen enables the user to edit and existing archetype or create a new archetype. The screen appears when the editor is loaded, the menu File > New is chosen or Ctrl-N is pressed

Edit an existing archetype

This portion of the screen is only available at start up. Press the button in the lower portion of the screen and choose the archetype using the file dialog.

Create a new archetype

This part of the screen is always visible - the cancel button is labelled 'exit' at start up as this exits the program.

1. Choose the authority

First choose the authority....at the moment only openEHR is a valid choice. As the openEHR model is very rich this should ensure that all the clinical requirements are captured.

2. Choose the model

Then choose the model - in openEHR this means choosing between the EHR and Demographic models. The former is for all clinical concepts, the latter is for demographic information in a demographic server. At present the editor will only work with the EHR model.

3. Choose the component

Then choose the component. In openEHR this involves choosing between a composition, a section or one of the three types of entry - observation, evaluation or instruction. These are the classes in the clinical reference model that can be archetyped. Review designing an archetype if you are not sure which component to use.

4. Name the archetype

Now name the archetype itself - the short concept label - using no spaces or hyphens ('-'). Use an underline instead if you want to have a space (e.g. represent blood pressure as 'blood_pressure').



Now press OK.
