Create a new archetype

This Editor aims to allow expert clinicians to design data constraints for concepts utilised within EHR systems. The outcome of sharing the archetypes produced is reuse of the data over the lifetime of the patient. To create a useful archetype it is important that you have a reasonable understanding of what an archetype is and how these differ from templates.

We strongly recommend that you take a little time to read the help on designing archetypes.

  1. Choose New from the File Menu or Press Ctrl-N. This will open the 'Start up' screen which enables you to choose the model, component and set the short concept name which identifies the archetype.

  2. Edit the archetype header

  3. Set the constraints on the core data

  4. Save the archetype
