Save an archetype

To save an archetype you can:

  • Choose the Menu > File > Save menu;
  • Hold down the Ctrl key and press 'S' - a shortcut.
  • Press the Save button on the toolbar (right side)

You will get the File dialog - you can rename the archetype - to prevent it over-writing the archetype. Do nothing if you want to overwrite the existing archetype - you will have to confirm this by pressing OK when asked.

Other formats

Archetypes can be saved in HTML (choose HTML on the drop down list in the save dialog box) and as a visual representation on the display tab exported as RTF or TEXT.
