Add or remove an element

An element is a name value pair - these are the leaf nodes of the data being described by the archetype. These can be added to list and tree structures. With table structures columns are added (or rows in rotated tables).

Add an element

An element may be added by pressing the [+] button and following the menu choices or by dragging the data type icon from the left onto the list, tree or table structures.

Once added type the name in the label or edit box - the constraints appropriate for this data type will appear on the right hand side of the screen.

To change the name, click on the label and then click again, as you would do in an Explorer window.

Remove an element

To remove an element, click on it and press the [-] button, or right click on the element and choose the 'Remove..' option.
