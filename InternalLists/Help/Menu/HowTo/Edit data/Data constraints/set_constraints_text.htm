Set constraints on text

There are a range of constraints that can be set on text.

Free text

Checking the Text radio button on the left sets the text constraint to free text. It is possible to add allowable text strings to this constraint by pressing the [+] button. These are not coded in any way. The [-] button removes the choice.

Internal codes

When choices are added as internal codes, they are coded and can be translated into all available languages. They can also be bound to external terminologies for importing and exporting information as required. An additon button [...] enables reuse of a term already in the archetype.

Terminology

Finally, the constraint may relate to an external terminology - again an acXXXX code is created which can link to that terminology and return the set of terms; the wording here only describes the sort of constraint. The actual constraint is described in the 'constraint binding'.
