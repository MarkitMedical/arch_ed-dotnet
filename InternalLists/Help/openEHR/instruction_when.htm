When to use an INSTRUCTION

An INSTRUCTION is used for data which determines what will happen in the future - what is planned.  An instruction has the following features:

  • It contains data about when the timing of health care events
  • It contains workflow information - has it been commenced, has it been cancelled or has it been completed
  • It contains information about the condition or conditions for the event to take place - based on date and time, or other values
  • It may contain information about what processing is required, particularly in relation to notifications

For example, a medication order is recorded as an instruction whereas a medication administration is recorded as an observation. The former has information about when it is to commence, what the frequency of administration is - the later has only information about one instance of this order being administered.
